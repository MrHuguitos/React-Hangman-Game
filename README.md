# Hangman Game (Jogo da Forca) 🎮

Bem-vindo ao **Hangman Game**! Este é um jogo clássico da forca, desenvolvido como parte dos meus estudos e aprimoramento em programação web focado no ecossistema React.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias e ferramentas:

- **React (v19)** - Biblioteca JavaScript para construção da interface de usuário (UI).
- **Vite** - Ferramenta de build super rápida para projetos front-end modernos.
- **JavaScript (ES6+)** - Lógica principal do jogo.
- **CSS3** - Estilização customizada e responsiva.
- **HTML5 (`<dialog>`)** - Utilização da tag semântica nativa para a criação de modais acessíveis (Sobre, Ajuda, Resultado).
- **React Router Dom** - Gerenciamento de rotas simples.

## 🌟 Funcionalidades

- **Sorteio Dinâmico:** As palavras do jogo são consumidas e sorteadas dinamicamente a partir de um arquivo de texto local (`palavras.txt`), garantindo que o jogo seja leve e não dependa de um servidor/banco de dados complexo.
- **Modais Nativos:** Janelas interativas utilizando as melhores práticas web modernas.
- **Ícones em SVG:** Interface limpa utilizando imagens vetoriais (SVG) para melhor performance e qualidade de renderização.
- **Feedback Visual:** Cores e mensagens intuitivas para indicar acertos, erros e letras repetidas.
- **Design Moderno:** Interface de usuário atrativa com transições de telas (Loading screen).

## 🛠️ Como rodar o projeto localmente

Se você quiser clonar e rodar este jogo na sua própria máquina, siga os passos abaixo:

### Pré-requisitos
- Ter o [Node.js](https://nodejs.org/) instalado.

### Passo a passo

1. Faça o clone deste repositório:
```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```

2. Entre na pasta do projeto:
```bash
cd Hangman-game
```

3. Instale as dependências necessárias:
```bash
npm install
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

5. O Vite fornecerá um link local (geralmente `http://localhost:5173/`). Basta abrir no seu navegador e jogar!
